# IronPal — Logo Design Image Generation Prompts

This document provides detailed AI image generation prompts for the top 3 selected logo concepts. Each logo has prompts covering:

1. **Primary mark** — the core logo on the primary brand background
2. **Color scheme variants** — the logo rendered for each of the three approved color schemes
3. **Application mockups** — the logo shown in real-world contexts (product, packaging, screen)

Reference documents:
- Logo specifications: [docs/logo-design-concepts.md](logo-design-concepts.md)
- Color schemes: [docs/color-schemes.md](color-schemes.md)

### Prompting Notes

- All prompts target **ChatGPT (DALL-E 3)**, **Midjourney v7**, or **FLUX 1.1 Pro**.
- For Midjourney, append `--ar 1:1` (square) or `--ar 16:9` (wide) as needed, and `--style raw` for cleaner vector-like output.
- For FLUX, prefix with `vector logo design,` for best results.
- Generate 4+ variants per prompt and select the cleanest execution.
- AI-generated logos will need manual refinement in Figma or Illustrator — treat these as high-fidelity starting references, not final art.

# Generation Workflow

1. **Generate primary marks first** (prompts 1A, 2A, 3A). Pick the cleanest execution for each.
2. **Generate color variants** (1B, 2B, 3B) using the same prompt structure with swapped colors.
3. **Generate the comparison sheet** to present all three side-by-side to the team.
4. **Generate application mockups** (1C, 2C, 3C) only for the logo(s) approved by the team after the comparison review.
5. **Refine in Figma/Illustrator.** Trace or rebuild the AI output as clean vector art. Finalize exact colors, spacing, and proportions. AI output is the reference — the final logo files should be hand-finished vectors.
